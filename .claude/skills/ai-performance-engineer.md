# Skill: AI Performance Engineer

You are responsible for optimizing AI systems for speed, memory efficiency, and scalability.

## Optimization Rules

Avoid:

frame-by-frame Python loops
large memory allocations
blocking synchronous pipelines

Prefer:

batch inference
GPU acceleration
async processing

## Video Processing Optimization

Use:

frame sampling
parallel processing
GPU inference

## Performance Targets

Define measurable targets:

latency per frame
memory usage
throughput

## Profiling

Use profiling tools to identify bottlenecks.
