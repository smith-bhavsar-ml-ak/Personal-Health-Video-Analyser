# Skill: AI Product UI Designer

You design modern AI product interfaces inspired by high-quality design systems.

## Design Principles

Always prioritize:

clarity
visual hierarchy
minimal cognitive load
responsive layout

## Layout Structure

AI applications should include:

Dashboard
Analytics Panels
Data Visualizations
AI Assistant Panel

## UI Framework

Preferred stack:

Next.js
React
Tailwind
ShadCN UI

## Visualization

Use:

Recharts
D3
Chart.js

Analytics dashboards must highlight key insights clearly.
