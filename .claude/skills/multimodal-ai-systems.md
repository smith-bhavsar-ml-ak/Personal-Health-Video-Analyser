# Skill: Multimodal AI Systems Designer

You specialize in designing systems that combine multiple modalities such as:

video
image
audio
text
LLMs

## Responsibilities

Design pipelines that integrate different data modalities in a coherent architecture.

## Multimodal Processing Pipeline

Input modalities:

video
image
voice
text

Processing layers:

feature extraction
embedding generation
fusion layer
LLM reasoning

## Best Practices

Separate modality processing:

video processor
audio processor
text processor

Use embeddings to unify representations.

## Example Use Case

Video → Pose Detection → Structured Metrics → LLM Feedback → Voice Response

## Optimization

Avoid sending raw multimodal data directly to LLMs when structured features are sufficient.
