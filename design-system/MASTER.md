# Personal Health Video Analyzer — Master Design System

Generated by: UI UX Pro Max skill
Project type: Health / Fitness AI SaaS Dashboard
Style: Clean Minimalist Dark Mode + Bento Grid
Stack: Next.js 14 + Tailwind CSS + ShadCN UI + Recharts

---

## 1. Design Identity

| Attribute     | Value                                          |
|---------------|------------------------------------------------|
| Style         | Minimalist Dark + Subtle Depth                 |
| Layout        | Sidebar + Bento Grid                           |
| Personality   | Precise, intelligent, clinical, motivating     |
| Anti-patterns | No gradients on backgrounds, no emoji icons    |
|               | No glassmorphism, no bright neons              |
|               | No border-radius > 12px on data cards          |

## 11. Motion / Animation

```css
/* Standard micro-interaction */
transition: all 150ms ease;

/* Page entrance */
animation: fadeInUp 200ms ease;

/* Skeleton loading */
animation: pulse 2s cubic-bezier(0.4,0,0.6,1) infinite;

/* Processing spinner */
animation: spin 1s linear infinite;
```

Rules:
- Never animate layout properties (width, height, top, left)
- Only animate: opacity, transform, color, background-color, border-color
- Respect prefers-reduced-motion

## 14. Anti-Patterns (Never Do)

- No gradient backgrounds on page or cards
- No border-radius > 16px on data cards
- No emoji as icons — use Lucide SVG
- No pure black (#000000) backgrounds — use #0A0A0F
- No white text on colored backgrounds (use brand-colored text on dimmed bg)
- No hover states that cause layout shift
- No fixed px font sizes below 14px
- No color as the only indicator of state (add icon or text)
